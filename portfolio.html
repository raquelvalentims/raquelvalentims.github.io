Sistema de Gerenciamento de Funcionários em Python com Listas, Dicionários e Funções

O projeto implementa um sistema de gerenciamento de funcionários em Python, utilizando lista de dicionários como estrutura de dados principal. Foram desenvolvidas funções específicas para cadastro, consulta (geral, por ID e por setor) e remoção, com uso de parâmetros, cópia de dicionários e tratamento de entradas inválidas. O programa principal possui um menu interativo com laço while e condicionais if/elif, além do controle de ID global incrementado automaticamente. O código inclui comentários explicativos e saída de console personalizada, demonstrando conceitos de estruturas de dados, modularização e controle de fluxo.

Sistema de Cobrança de Camisetas em Python com Funções e Tratamento de Erros

O projeto implementa um sistema de cálculo de vendas de camisetas em atacado, utilizando funções modulares para escolha de modelo, quantidade e frete. O programa aplica regras condicionais de desconto progressivo, valida entradas com laços de repetição e blocos try/except e retorna os valores correspondentes via return. No código principal, o cálculo do total a pagar é estruturado com base nos parâmetros retornados, garantindo robustez e flexibilidade. O sistema inclui comentários explicativos, tratamento de erros e mensagens de saída personalizadas, contemplando diferentes cenários de teste.

Aplicativo de Vendas de Marmitas em Python com Estruturas de Controle e Acumulador

O projeto implementa um sistema de pedidos de marmitas, utilizando entradas validadas para sabor (BA/FF) e tamanho (P/M/G), com mensagens de erro em casos inválidos. A lógica de preços é estruturada com condicionais aninhados (if/elif/else) para tratar todas as combinações possíveis. O programa emprega um acumulador de valores para somar múltiplos pedidos, controlado por laços while, break e continue que permitem repetição ou finalização. O fluxo interativo garante robustez na entrada de dados, enquanto comentários explicativos e mensagens personalizadas tornam a interface mais clara para o usuário.
